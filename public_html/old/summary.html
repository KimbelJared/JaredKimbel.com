Architecture Summary

Sometimes called the von Neumann model, the von Neumann architecture is a computer architecture designed by John von Neumann. The architecture consists of a processing unit, a control unit, a memory unit and input/output devices.

The design is simpler than the Harvard architecture, but also very similer. Both architectures use a stored-program system, but one has a dedicated set of address and data buses.
